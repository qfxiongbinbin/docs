## 账户管理

### 用户注册

<Steps>
  1. 访问慧衍平台首页，点击右上角的"注册"按钮
  2. 填写注册信息
    - 用户名（必填）
    - 邮箱地址（必填）
    - 密码（必填，至少8位）
    - 确认密码
  3. 阅读并同意用户协议
  4. 点击"注册"完成账号创建
</Steps>

<Tip>
建议使用企业邮箱注册，这将有助于后续的团队协作。
</Tip>

### 用户登录

<CodeGroup>
  ```bash 普通登录
  1. 输入用户名/邮箱
  2. 输入密码
  3. 点击"登录"
  ```

  ```bash 扫码登录
  1. 点击"扫码登录"
  2. 使用慧衍APP扫描二维码
  3. 确认登录
  ```
</CodeGroup>

### 找回密码

<AccordionGroup>
  <Accordion title="通过邮箱找回">
    1. 点击登录页的"忘记密码"
    2. 输入注册邮箱
    3. 获取验证码
    4. 设置新密码
  </Accordion>

  <Accordion title="通过管理员重置">
    如果无法通过邮箱找回，请联系您的团队管理员重置密码。
  </Accordion>
</AccordionGroup>

## 团队管理

### 团队设置

<Tabs>
  <Tab title="创建团队">
    <Steps>
      1. 进入工作台，点击"创建团队"
      2. 填写团队信息：
         - 团队名称
         - 团队描述
         - 团队logo（可选）
      3. 设置团队权限
      4. 确认创建
    </Steps>
  </Tab>
  
  <Tab title="团队配置">
    管理员可以配置以下内容：
    - 基本信息设置
    - 权限策略
    - 资源配额
    - 团队公告
  </Tab>
</Tabs>
